# 十进制进制转换工具 (Base Conversion Tool)

## 项目简介

这是一个用 C 语言实现的十进制进制转换工具，可将输入的十进制数转换为：
- **二进制原码** - 符号位 + 数值位表示
- **二进制反码** - 负数各位取反（符号位不变）
- **二进制补码** - 反码加1（计算机内部表示）
- **十六进制数** - 16进制表示
- **八进制数** - 8进制表示

## 业务逻辑说明

### 1. 二进制原码 (Binary Original Code)
- **正数**: 符号位为 0，后续为数值的二进制表示
- **负数**: 符号位为 1，后续为数值绝对值的二进制表示

示例：
```
5 的原码：  01010101
-5 的原码： 11010101
```

### 2. 二进制反码 (Binary Ones' Complement)
- **正数**: 反码 = 原码（不变）
- **负数**: 反码 = 原码符号位不变，其他各位取反

示例：
```
-5 的反码：11101010 （原码11010101，除符号位外取反）
```

### 3. 二进制补码 (Binary Two's Complement)
- **正数**: 补码 = 原码（不变）
- **负数**: 补码 = 反码 + 1

示例：
```
-5 的补码：11101011 （反码11101010 + 1）
```

### 4. 进制转换
- **十六进制**: 每4位二进制表示一位十六进制数字
- **八进制**: 每3位二进制表示一位八进制数字

## 核心模块说明

### converter.h (头文件)
定义了所有对外接口：
- `ConversionResult` - 存储转换结果的结构体
- `convert_decimal()` - 主转换函数
- 各种转换的辅助函数

### converter.c (实现文件)
包含所有核心业务逻辑：
- `decimal_to_binary_original()` - 十进制转二进制原码
- `binary_ones_complement()` - 原码转反码
- `binary_twos_complement()` - 反码转补码
- `decimal_to_hexadecimal()` - 十进制转十六进制
- `decimal_to_octal()` - 十进制转八进制
- `print_conversion_result()` - 格式化输出结果

### main.c (主程序)
处理用户交互和菜单逻辑：
- 交互式菜单界面
- 输入验证
- 示例演示
- 使用说明

## 技术细节

- **语言**: C (C99 标准)
- **编译器**: GCC (MinGW on Windows, GCC on Linux)
- **数据类型**: 使用 `int32_t` 确保一致的整数大小
- **缓冲区管理**: 使用固定大小缓冲区存储各进制结果
- **错误处理**: 返回错误码检查转换是否成功
